# Speech Assignment

Brief workspace for three tasks:

- `q1/` Signal processing pipeline (manual MFCC, leakage/SNR, voiced-unvoiced, phone alignment)
- `q2/` Speaker verification experiments (base and improved models + evaluation)
- `q3/` Privacy/fairness audit and offline demo package

## Quick Run

From each folder, run the main scripts:

- `q1`: `python mfcc_manual.py`, `python leakage_snr.py`, `python voiced_unvoiced.py`, `python phonetic_mapping.py`
- `q2`: `python train.py`, `python eval.py`, `python improved_exp.py`
- `q3`: `python audit.py`, `python pp_demo.py`, `python train_fair.py`

See each subfolder README for full setup and detailed instructions.
