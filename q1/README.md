# Q1: Multi-Stage Cepstral Feature Extraction and Boundary Detection

## Deliverables in this folder

1. `mfcc_manual.py` - handcrafted MFCC/Cepstrum engine
2. `leakage_snr.py` - spectral leakage and SNR comparison for windows
3. `voiced_unvoiced.py` - voiced/unvoiced boundary detection from cepstral regions
4. `phonetic_mapping.py` - Hugging Face alignment-style mapping and RMSE
5. `q1_report.pdf` - generated by `generate_report.py`
6. `audio_manifest.csv` - audio list used in experiments

## Dependencies

```bash
pip install numpy scipy pandas matplotlib soundfile torch transformers
```

## Run End-to-End

```bash
cd /data2/autoNav/akshat/speech/q1

# Pick audio from manifest
AUDIO=path/to/audio/file

python mfcc_manual.py --audio "$AUDIO" --out_dir output/mfcc
python leakage_snr.py --audio "$AUDIO" --out_dir output/leakage_snr
python voiced_unvoiced.py --audio "$AUDIO" --out_dir output/voiced_unvoiced

# Downloads model weights on first run
python phonetic_mapping.py --audio "$AUDIO" --out_dir output/phonetic_mapping

python generate_report.py --base_dir . --out_pdf q1_report.pdf
```

## Notes

- `phonetic_mapping.py` defaults to `facebook/wav2vec2-base-960h` for phone-like tokens.
- If no GPU is available, the script automatically runs on CPU.
- RMSE is reported in both seconds and milliseconds.
